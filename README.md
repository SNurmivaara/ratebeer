## Useful commands

```
# Update gems
bundle update

# Install
bundle install

# Run server
rails s

# Run console
rails c

# Rubocop test and fix automatically fixable
rubocop -A

# Deploy to fly.io
fly deploy

# Check logs on fly.io
fly logs

# Access ruby console on fly.io
fly ssh console
/app/bin/rails c

```
